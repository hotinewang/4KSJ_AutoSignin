## 2023-9-19关于中文用户名无法登录的反馈
接用户反馈，中文用户名无法登录，应该是post请求编码的问题，捣鼓了以下，最后没能从代码方面解决这个问题-_-|||。但是，把中文用户名修改成论坛id（在论坛上登录之后可以看到），脚本即可成功签到。（理论上换成邮箱地址也可以。）
